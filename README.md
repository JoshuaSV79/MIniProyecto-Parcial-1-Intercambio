# MIniProyecto-Parcial-1-Intercambio
Mini Proyecto dedicado a la evaluacion del primer parcial de Tecnologias Web, con el proposito de implementar y usar una de las dos librerias BootStrap o Tailwind.

## Tecnologias Usadas 
* CSS nativo
* JavaScript
* HTML

!note [Este proyecto no incluye un bakend unicamente un frontend]
